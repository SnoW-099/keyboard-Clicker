# Keyboard Clicker

Clicker de teclado inspirado en paneles de progresion tipo arcade, con un teclado central clicable tecla por tecla.

## Desarrollo

Instala dependencias:

```bash
npm install
```

Arranca el servidor local:

```bash
npm run dev
```

Build para Vercel:

```bash
npm run build
```

## Incluye

- Teclado central con todas las teclas clicables con raton.
- Tambien funciona pulsando teclas fisicas del teclado.
- Sonido mecanico distinto por zona de tecla.
- Clics totales, reputacion, niveles y combos.
- Mejoras manuales y generadores automaticos.
- Coleccion desbloqueable de teclados.
- Renacimiento con multiplicador permanente.
- Eventos aleatorios, efectos, sonidos y guardado local.
